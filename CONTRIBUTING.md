# Contributing

```sh
# Install deps
npm install

# Compile the plugin
npm run compile

# Or compile the plugin every time a file is saved
npm run compile-watch
```

In VS Code press F5 from within the project to open an instance of VS Code
with the local version of the extension loaded.
